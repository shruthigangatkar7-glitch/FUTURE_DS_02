Customer Retention and Churn Analysis Dashboard

📌 Project Overview
This project focuses on analyzing customer retention and churn behavior using **Power BI** and a structured customer dataset. The goal of this dashboard is to identify the factors influencing customer churn and provide business insights that can help improve customer retention strategies.

The project uses a CSV dataset containing customer details such as tenure, contract type, payment methods, monthly charges, total charges, and churn status.

📊 Dashboard Highlights
The Power BI dashboard provides interactive visualizations for:
- Churn by Payment Method
- Churn by Tenure
- CustomerId by Payment Method and Churn
- CustomerId by Monthly Charges and Churn

🛠️ Tools & Technologies Used

- **Power BI Desktop**
- **CSV Dataset**
- Data Cleaning & Transformation using Power Query
- Data Visualization & Business Intelligence

## 🎯 Project Objective

To help businesses understand customer behavior, reduce churn rate, and improve long-term customer retention through data-driven decisions.
